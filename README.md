# Kanban Fundamentals

A training program covering the core principles and practices of the Kanban method.

## Overview

This program introduces the fundamentals of Kanban, a workflow management method for defining, managing, and improving services that deliver knowledge work.

## Topics

- What is Kanban?
- Core principles and practices
- Visualizing workflow
- Limiting work in progress (WIP)
- Managing flow
- Making policies explicit
- Implementing feedback loops
- Improving collaboratively

## Getting Started

Follow the modules in order to build a solid understanding of Kanban fundamentals.
